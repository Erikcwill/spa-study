
# Projeto de Estudo SCSS

Este é um projeto de estudo em que um arquivo HTML pré-existente foi fornecido e o objetivo principal é aprimorar as habilidades em SCSS. O projeto se concentra principalmente na aplicação de estilos usando SCSS.

## Tecnologias utilizadas

As seguintes tecnologias foram utilizadas neste projeto:

-   HTML
-   SCSS

Não foi necessário o uso do Node.js para executar este projeto. Basta abrir o arquivo `index.html` em um navegador para visualizar a página.

## Objetivo do projeto

O objetivo deste projeto é aprimorar as habilidades em SCSS e aplicar estilos no arquivo HTML fornecido. Foram feitas algumas modificações no HTML, mas o foco principal é aprimorar as habilidades em SCSS.

## Como utilizar o projeto

Para utilizar este projeto, basta baixar os arquivos em seu computador e abrir o arquivo `index.html` em um navegador. Todos os estilos foram aplicados por meio do arquivo `style.scss`, que é compilado em um arquivo CSS.

## Considerações finais

Este projeto foi uma oportunidade para aprimorar habilidades em SCSS e praticar a aplicação de estilos em um arquivo HTML pré-existente. O objetivo foi alcançado e o projeto foi concluído com sucesso.
